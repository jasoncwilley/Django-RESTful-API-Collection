RobotHash


A cryptographic hash function is a hash function which takes an input (or 'message') and returns a fixed-size alphanumeric string. Some standard applications that employ hash functions include authentication, message integrity (using an HMAC (Hashed MAC)), message fingerprinting, data corruption detection, and digital signature efficiency.

Robohash is a web service that makes it easy to provide unique image of a robot, monster or kitten for any given string of characters that can be displayed on a webpage and help ensure data integrity. Go ahead and give it try. Use the forms first column below to create a Robohash and check it using the corresponding form under the second column. If you enter the same string in both forms it will return the same pictures.
